Slow Syscalls ≥ 1ms by Process (whole run)
Total blocked time in syscalls that took at least 1ms, for containerd / containerd-shim / runc / kubelet. This is off-CPU blocking time, which CPU profilers mostly miss. Idle waits (futex, epoll/poll, sleeps, child-process waits) are excluded: blocking there is a thread waiting for work, not work waiting for the kernel.
